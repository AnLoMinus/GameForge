# 💡 Logic Universe – Riddles & Cyber Puzzles

חידות חשיבה, קודים ופאזלי UI עם אווירת Matrix מוזהב.

## מבנה
- `puzzles/` – קבצי חידה (Markdown/JSON) עם פתרונות מוסתרים.
- `ui/` – דוגמאות ממשק לחידות אינטראקטיביות.

## פאזל פתיחה: "Gate Cipher"
- **רמז:** לכל Realm יש מספר (1–7). סדר את המספרים לפי האייקון שעל הקלף "Gate Pulse" כדי לקבל קוד 1724635.
- **שימוש:** הקוד פותח אזור בונוס במשחק דפדפן (ראו `browser-games/`).
