# 🌐 Web Adventures – Browser Games Hub

משחקי דפדפן קלים להרצה (HTML/CSS/JS), מותאמים ל־GitHub Pages.

## מבנה
- `html/` – קבצי דפי משחק.
- `styles/` – עיצוב משותף למשחקים.
- `scripts/` – לוגיקת משחק, ניקוד וקלט משתמש.

## Mini Game: "Spark Runner"
- **סוג:** משחק התחמקות ואיסוף אנרגיה.
- **איך להריץ:** פתחו את [`html/spark-runner.html`](./html/spark-runner.html) בדפדפן מקומי.
- **לופ משחק:** הוזזת שחקן עם חצי מקלדת, איסוף Orbs (ניקוד), הימנעות ממטאורים.
- **קשר ליקום:** קודים מחידות ב־`logic-puzzles/` יכולים לפתוח מצבי קושי חדשים (קוד ברירת מחדל: 1724635).

ניתן להוסיף משחקים נוספים ע"י יצירת קובץ HTML חדש + סקריפט/סטייל ייעודיים.
