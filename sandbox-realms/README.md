# 🧱 Creator Sandbox – Worlds & Simulations

מקום לבנות סימולציות, סביבות ניסוי וכלי בנייה המתממשקים לכל יתר הממלכות.

## מבנה
- `worlds/` – תיאורי עולמות, סביבות ואבני בניין (JSON/Markdown).
- `building-tools/` – מחוללי מפות, עורך בלוקים, או מדריכי אינטגרציה.

## Mini-Sim: "Spark Garden"
- **רעיון:** גינה דיגיטלית שמגיבה למוזיקה מה־`music-games/` ומגדילה פרחים כאשר שחקנים משיגים Spark Points.
- **שימוש:** בסיס ליצירת sandbox שמקבל נתוני התקדמות מכל Realm.
