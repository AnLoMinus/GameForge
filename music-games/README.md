# 🎧 Beat Sparks – Rhythm & Reaction

משחקי קצב עם פולסים מוזהבים וסינרגיה בין אודיו לאירועים במשחקים אחרים.

## מבנה
- `tracks/` – לופים, דמואים וקובצי תזמור (לשילוב עתידי).
- `rhythm-engine/` – סקריפטים או תרשימים למנוע תזמון (BPM, סנכרון קלט).

## רעיון פתיחה: "Resonance Lane"
- **Gameplay:** נתיבי אור מגיעים מהחלל; לחיצה בזמן הנכון טוענת אנרגיה.
- **Cross-Over:** טעינת אנרגיה מושקת ל־`card-realms` כ־Spark Points במשחק שולחני משולב.
