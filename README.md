## ✨ TL;DR – מה זה GameForge?

GameForge הוא **בית אחד לכל המשחקים**:
🎴 משחקי קלפים • 🌐 משחקי דפדפן • 🧭 RPG • 🥷 אקשן והרפתקאות • 💡 חידות לוגיות • 🎧 משחקי מוזיקה וקצב • 🧱 Sandbox יצירה ובנייה.

זהו **יקום מרכזי** שבו אפשר:

* לבנות רעיונות למשחקים מאפס
* לפתח מנועי משחק פשוטים ומתקדמים
* לאחד את כל פרויקטי המשחקים תחת שפה אחת: **SparKing / GameForge Style**
* ליצור גשרים בין קלפים, אתרי GitHub Pages, מוזיקה, ועלילות RPG לעולם אחד מתפתח.

> 🎯 מטרה: להפוך את כל הברקים וההברקות שלך — ל**עולמות משחק חיים**.

## 🚀 Workflow – איך מפתחים משחק ב־GameForge

1️⃣ **Spark – רעיון מהבזק** 💡

* משפט אחד: מה השחקן עושה ולמה זה כיף?

2️⃣ **Shape – הגדרת המכניקה** 🎛️

* מה התנועות? מה האתגרים? מה הפידבקים?

3️⃣ **World – בניית עולם** 🌍

* אווירה, צבעים, אנרגיה, חוקי יקום.

4️⃣ **Prototype – אב טיפוס** 🧪

* גרסה ראשונית, גם אם “מכוערת” — העיקר משחקת.

5️⃣ **Playtest – בדיקות חוויה** 🎯

* לזהות: איפה כיף? איפה משעמם? איפה מסובך?

6️⃣ **Polish – ליטוש SparKing** ✨

* אפקטים, אנימציות, סאונד, טקסטים.

7️⃣ **Release – שחרור לעולם** 🚀

* מחברים ל־GitHub Pages / קלפים / וידאו / מוזיקה.

## 🎴 Gate 01 – Card Realms (Core Template)

בתוך `card-realms/` יוצרים סטנדרט לכל משחק קלפים:

* 🧾 `card-realms/README.md` – תיאור המשחקים, חוקים בסיסיים, פילוסופיה.
* 🧩 `prototypes/` – סקיצות למבנה קלפים (JSON / Markdown).
* 🎨 `assets/` – תמונות, מסגרות, רקעים בסגנון SparKing.
* 📜 `docs/` – חוקים מורחבים, סטים (Decks), עולמות.

בכל משחק קלפים:

* **Card ID** (לדוגמה: `GF-CR-01`)
* **Title**
* **Type** (Attack / Support / Realm / Event…)
* **Effect**
* **Lore** (שורה קצרה של סיפור/חכמה)

---

## 🌐 Gate 02 – Web Adventures

ב־`browser-games/` נבנים משחקי דפדפן:

* דפים סטטיים ריצים על GitHub Pages.
* JS בסיסי לאינטראקציה (תנועה, קליקים, ניקוד).
* התאמה לעולמות קיימים: SparKing, Exit the Matrix, ParallelPranks, DreamHints ועוד.

---

## 🧭 Gate 03 – RPG Realities

בתוך `rpg-worlds/`:

* `lore/` – סיפורי רקע, יקומים, צירי זמן.
* `quests/` – משימות, שלבים, תיקוני דרך.
* `characters/` – דמויות עם נתונים (Health, Energy, Wisdom, Spark Level).

כאן מגדירים גם **“Path of Ascension”** – מסע התקדמות לפי ספירות / שערים / קלפים.
