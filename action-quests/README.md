# 🥷 Action Quests – Fast Missions & Boss Flows

פלטפורמרים, קרבות ובוס-ראשים עם וייב SparKing.

## מבנה
- `levels/` – תרשימי שלב, מפות ASCII או JSON.
- `enemies/` – נתוני אויבים, AI sketch, טבלאות נזק.

## דוגמה: "Beacon Sprint"
- **רעיון:** רוץ בשלושה אזורי פלטפורמה והציב Beacon לפני שהאנרגיה מתרוקנת.
- **מכאניקה:** Dash (סטמינה), Wall-Jump, Charge Shot.
- **סיום:** אם כל Beacon פועלים תוך 90 שניות – פותחים Gate שמעניק בונוס ב־`card-realms`.

שמרו על סינרגיה עם הלור ב־`rpg-worlds` לחוויה אחודה.
